Built-in Data Types

Python provides a rich set of built-in data types that are essential for performing various operations. Here are a few important ones:

  • Integer: The 'int' data type represents whole numbers without fractional parts. It supports arithmetic operations such as addition, subtraction, multiplication, and division.
    Example:
                                    
                                    x = 5
                                    y = 3
                                    z = x + y
                                    print(z) # Output: 8
  • String: The 'str' data type represents a sequence of characters enclosed in single quotes ('') or double quotes (""). Strings are immutable, meaning they cannot be changed after the are created.
    Example:
                                    
                                    message = "Hello, world!"
                                    print(message) # Output: Hello, world!
  • List: The 'list' data type is an ordered collection of elements enclosed in square brackets []. Lists can contain elements of different data types and can be modified using various built-in methods such as 'append()', 'extend()', and 'pop()'.
    Example:
                                    
                                    fruit = ['apple', 'banana', 'orange']
                                    fruit.append('grape')
                                    print(fruits) # Output: ['apple', 'banana', 'orange', 'grape']
Control Flow

Control flow statements allow you to dictate the order in which your code is executed. They help you make decisions, iterate over elements, and control program flow. Here are a few important control flow statements:

  • If-else: The 'if-else' statement allows you to execute a block of code based on a condition. If the conditions is true, the code inside the 'if' block is executed; otherwise, the code inside the 'else' block is executed.
    Example:
                                    
                                    age = 18
                                    if age >= 18:
                                        print("You are an adult.")
                                    else:
                                        print("You are a minor.")
  • For Loop: The 'for' loop is used to iterate over a sequence (such as a string, list, or tuple) or other iterable objects. It allows you to perform a set of statements repeatedly for each item in the sequence.
    Example:
                                    
                                    fruits = ['apple', 'banana', 'orange']
                                    for fruit in fruits:
                                        print(fruit)
  • While Loop: The 'while' loop is used to execute a block of code repeatedly as long as condition is true. It is useful when you want to repeat a set of statements until a certain condition is met.
    Example:
                                    
                                    count = 0
                                    while count > 5:
                                        print(count)
                                        count += 1
File Handling

Python provides various functions and methods to handle files. Here are a few file handling operations you should be familiar with:

  • Opening a File: The 'open()' function is used to open a file in Python. It takes two arguments: the file name and the mode in which the file should be opened (e.g., read mode, write mode, append mode).
    Example:
                                    
                                    file = open('data.txt', 'r')
  • Reading from a File: The 'read()' method is used to read the contents of a file. It reads the entire file as a single string or can read a specific number of characters at a time using the 'read(n)' method.
    Example:
                                    
                                    file = open('data.txt', 'r')
                                    content = file.read()
                                    print(content)
  • Writing to a File: The 'write()' method is used to write data to a file. It takes a string as an argument and writes it to the file. If the file does not exist, it will be created.
    Example:
                                    
                                    file = open('data.txt', 'w')
                                    file.write('Hello, world!')
                                    file.close()
Regular Expressions

Regular Expressions provide a powerful way to search, match, and manipulate text patterns in Python. Here are a few essential concepts related to regular expressions:

  • Pattern Matching: A regular expression, also known as a pattern is a sequence of characters that defines a search pattern. Python's 're' module provides functions such as 'search()', 'match()', and 'findall()' for pattern matching.
    Example:
                                    
                                    import re

                                    text = "Hello, world!"
                                    pattern = r"Hello"
                                    match = re.search(pattern, text)
                                    if match:
                                        print("Pattern found.")
  • Metacharacters: Metacharacters are characters with special meanings in regular expressions. Some commonly used metacharacters include '.' (matches any character), '*' (matches zero or more occurrences), and '+' (matches one or more occurences).
    Example:
                                    
                                    import re

                                    text = "The quick brown fox jumps over the lazy dog."
                                    pattern = r".o."
                                    matches = re.findall(pattern, text)
                                    print(matches) # Output: ['fox', 'dog']
  • Character Classes: Character classes allow you to match a specific set of characters. For example, '[0-9]' matches any digit, '[a-zA-Z]' matches any alphabetic character, and '[^0-9]' matches any character except a digit.
    Example:
                                    
                                    import re

                                    text = "The password must contain at least one uppercase letter."
                                    pattern = r"[A-Z]"
                                    match = re.search(pattern, text)
                                    if match:
                                        print("Uppercase letter found.")
Database Connectivity

Python offers various libraries and modules to connect and interact with databases. Here are a few commonly used modules for database connectivity:

  • SQLite3: Then 'sqlite3' module procides a simple and lightweight way to interact with SQLite databases. It allows you to create, modify, and query databases using SQL statements.
    Example:
                                    
                                    import sqlite3

                                    conn = sqlite3.connect('example.db')
                                    cursor = conn.cursor()
                                    cursor.execute('''CREATE TABLE users (id INT PRIMARY KEY, name TEXT)''')
                                    conn.commit()
                                    conn.close()
  • MySQL Connector: The 'mysql-connector-python' module enables Python programs to access MySQL databases. It provides an interface for connecting to MySQL servers, executing SQL queries, and managing transactions.
    Example:
                                    
                                    import mysql.connector

                                    conn = mysql.connector.connect(user='username', password='password', host='localhost', database='mydb')
                                    cursor = conn.cursor()
                                    cursor.execute('''SELECT * FROM customers''')
                                    rows = cursor.fetchall()
                                    for row in rows:
                                        print(row)
                                    conn.close()
  • PostgreSQL Connector: The 'psycopg2' module is a PostgreSQL adapter for Python. It allows you to connect to PostgreSQL databases, execute SQL statements, and retrieve results efficiently.
    Example:
                                    
                                    import psycopg2

                                    conn = psycopg2.connect(database="mydb", user="myuser", password="mypassword", host="localhost", port="5432")
                                    cursor = conn.cursor()
                                    cursor.execute('''SELECT * FROM employees''')
                                    rows = cursor.fetchall()
                                    for row in rows:
                                        print(row)
                                    conn.close()
Exception Handling

Exception handling allows you to catch and handle errors or exceptions that may occur during the execution of your code. It helps in maintaining the flow of the program and providing appropriate error messages. Here are a few important concepts related to exception handling:

  • Try-Except: The 'try-except' block is used to handle exceptions. The code insde the 'try' block is executed, and if any exception occurs, it is caught and handled by the corresponding 'except' block
    Example:
                                    
                                    try:
                                        result = 10 / 0
                                        print(result)
                                    except ZeroDivisionError:
                                        print("Cannot divide by zero.")
  • Raise: The 'raise' statement is used to explicitly raise an exception. It allows you to create custom exceptions or propagate existing exceptions to higher levels of the program.
    Example:
                                    
                                    def validate_age(age):
                                        if age < 0:
                                            raise ValueError("Age cannot be negative.")
                                        elif age < 18:
                                            raise ValueError("Must be at least 18 years old.")
                                        else:
                                            print("Age is valid.")

                                    try:
                                        validate_age(15)
                                    except ValueError as e:
                                        print(str(e))
  • Finally: The 'finally' block is used to specify code that should be executed regardless of whether an exception occurred or not. It is typically used to perform cleanup operations, such as closing files or releasing resources.
    Example:
                                    
                                    file = None
                                    try:
                                        file = open("data.txt", "r")
                                        content = file.read()
                                        print(content)
                                    except FileNotFoundError:
                                        print("File not found.")
                                    finally:
                                        if file:
                                            file.close()
Object-Oriented Programming (OOP)

Object-Oriented Programming (OOP) is a programming paradigm that emphasizes the concept of objects and classes. It provides a way to structure code by grouping related data and functions together. Here are a few important concepts related to OOP in Python:

  • Class: A class is a blueprint or template for creating objects. It defines the properties (attributes) and behaviors (methods) that objects of that class will have.
    Example:
                                    
                                    class Car:
                                        def __init__(self, brand, color):
                                            self.brand = brand
                                            self.color = color

                                        def start_engine(self):
                                            print("Engine started.")

                                    my_car = Car("Toyota", "Red")
                                    print(my_car.brand)  # Output: Toyota
                                    my_car.start_engine()  # Output: Engine started.
  • Inheritance: Inheritance is a mechanism that allows you to define a new class based on an existing class. The new class inherits the attributes and methods of the existing class, allowing code reuse and creating a hierarchy of classes.
    Example:
                                    
                                    class ElectricCar(Car):
                                        def __init__(self, brand, color, battery_capacity):
                                            super().__init__(brand, color)
                                            self.battery_capacity = battery_capacity

                                        def charge_battery(self):
                                            print("Battery charged.")

                                    my_electric_car = ElectricCar("Tesla", "Blue", 75)
                                    print(my_electric_car.color)  # Output: Blue
                                    my_electric_car.start_engine()  # Output: Engine started.
                                    my_electric_car.charge_battery()  # Output: Battery charged.
  • Polymorphism: Polymorphism allows objects of different classes to be used interchangeably based on their common interface. It provides flexibility and allows the same code to work with different objects.
    Example:
                                    
                                    def describe_vehicle(vehicle):
                                        print(f"Brand: {vehicle.brand}, Color: {vehicle.color}")

                                    my_car = Car("Toyota", "Red")
                                    my_electric_car = ElectricCar("Tesla", "Blue", 75)

                                    describe_vehicle(my_car)  # Output: Brand: Toyota, Color: Red
                                    describe_vehicle(my_electric_car)  # Output: Brand: Tesla, Color: Blue
Web Scraping

Web scraping is the process of extracting data from websites. Python provides powerful libraries and tools for web scraping, making it easier to retrieve and manipulate data from web pages. Here are a few important concepts related to web scraping:

  • Requests: The 'requests' library allows you to send HTTP requests and retrieve the content of web pages. It provides functions such as 'get()' and 'post()' for making HTTP requests.
    Example:
                                    
                                    import requests

                                    response = requests.get("https://www.example.com")
                                    print(response.status_code)  # Output: 200
                                    print(response.text)  # Output: HTML content of the page
  • Beautiful Soup: The 'BeautifulSoup' library is used for parsing HTML and XML documents. It provides a simple interface to navigate and search the parsed data, making it easy to extract specific elements or data from web pages.
    Example:
                                    
                                    from bs4 import BeautifulSoup

                                    html = "<html><body><h1>Hello, world!</h1></body></html>"
                                    soup = BeautifulSoup(html, "html.parser")
                                    heading = soup.find("h1")
                                    print(heading.text)  # Output: Hello, world!
  • XPath and CSS Selectors: XPath and CSS selectors are used to locate specific elements within an HTML document. They provide a concise and powerful way to navigate and extract data from the document.
    Example:
                                    
                                    from lxml import etree
                                    from cssselect import CSSSelector

                                    html = "<html><body><ul><li>Item 1</li><li>Item 2</li></ul></body></html>"
                                    tree = etree.fromstring(html)
                                    selector = CSSSelector("li")
                                    items = selector(tree)
                                    for item in items:
                                        print(item.text)  # Output: Item 1, Item 2
